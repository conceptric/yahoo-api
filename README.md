# Yahoo::Api

I've been using Yahoo! Query Language to obtain quoted asset data from Yahoo! Finance. This project aims to simplify the construction and use of YQL request URIs when used from other Ruby projects.

## Installation

Add this line to your application's Gemfile:

    gem 'yahoo-api'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install yahoo-api
